To generate PNG icons from the SVG files:

  1. Open icon-192.svg in a browser
  2. Screenshot or use an online SVG-to-PNG converter
  3. Save as icon-192.png and icon-512.png in this folder
  4. Or use: npx svg2png-cli icon-192.svg -o icon-192.png -w 192 -h 192

For now the PWA will work with SVG fallback on most modern browsers.
